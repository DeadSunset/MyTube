# Импорт YouTube URL в MyTube (через YouTube API)

## 1) Точечный импорт одного видео

1. Откройте видео в MyTube.
2. Вставьте `https://www.youtube.com/watch?v=...`.
3. Нажмите **«Импорт по URL»**.

Загрузятся: просмотры, лайки, до 50 комментариев и до 10 ответов к каждому.

## 2) Автопарсинг по конкретному каналу/папке

Теперь автопарсинг делается **внутри выбранной папки библиотеки**:

1. В левой панели выберите нужную папку (канал).
2. Вверху библиотеки появится строка: **«Парсинг только для папки ...»**.
3. Вставьте URL YouTube-канала (опционально, но желательно).
4. Нажмите:
   - **«Обновить первые 20 из папки»** — быстрый тест;
   - **«Автопарсить текущую папку»** — полный прогон только по этой папке.

## 3) Если видите `YouTube API error: 403`

Это почти всегда проблема настроек ключа в Google Cloud:

- используйте именно **API Key** (не OAuth Client ID/Secret);
- включите `YouTube Data API v3`;
- проверьте quota;
- проверьте ограничения ключа (HTTP referrer / application restrictions).

Сообщение про OAuth consent screen к этому приложению не относится: здесь нужен именно API key.
