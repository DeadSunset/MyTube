# Импорт YouTube URL в MyTube (через YouTube API)

Импорт работает по кнопке в интерфейсе:
- вставляете URL видео;
- нажимаете «Импорт по URL»;
- приложение подтягивает просмотры, лайки, до 50 комментариев и до 10 ответов к каждому комментарию.

## Что нужно сделать пользователю

1. Открыть видео в MyTube.
2. Вставить `https://www.youtube.com/watch?v=...`.
3. Нажать **«Импорт по URL»**.

## Автопарсинг всей библиотеки

В профиле доступны две кнопки:
- **«Обновить первые 20 из списка»** — быстрый тестовый прогон;
- **«Автопарсить всю библиотеку»** — полный прогон по всей библиотеке.

Также в профиле можно указать:
- **Канал в библиотеке** — текстовый фильтр (например имя канала/папки у ваших локальных видео);
- **URL этого канала на YouTube** — чтобы искать только внутри нужного канала на стороне YouTube.

Важно: это ограничение применяется только к выбранному каналу в библиотеке, а не ко всем 15000 видео.

Во время процесса показывается прогресс-бар.

## Если видите `YouTube API error: 403`

Это означает, что запросы блокируются настройками ключа в Google Cloud.
Проверьте:
- используете именно **API Key**, а не OAuth Client ID/Secret;
- включен `YouTube Data API v3`;
- не исчерпана quota;
- корректно настроены ограничения ключа (HTTP referrer/application restrictions).

Сообщение про OAuth consent screen обычно появляется, когда в проекте используется OAuth-клиент. Для этого приложения нужен API Key.

## Что импортируется

- просмотры;
- лайки;
- до 50 верхнеуровневых комментариев с никнеймами авторов;
- до 10 первых ответов к каждому комментарию (если ответы доступны через API);
- превью, название и канал.
