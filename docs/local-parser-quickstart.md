# Импорт YouTube URL в MyTube (без сложной настройки)

Теперь базовый сценарий работает **без запуска локального сервера**:
- вставляете URL видео;
- нажимаете «Импорт по URL»;
- приложение пробует получить просмотры, лайки и до 50 комментариев и до 10 ответов к каждому комментарию через YouTube Data API прямо из браузера.

## Что нужно сделать пользователю

1. Открыть видео в MyTube.
2. Вставить `https://www.youtube.com/watch?v=...`.
3. Нажать **«Импорт по URL»**.

На этом всё.

## Когда нужен локальный парсер

Локальный парсер теперь **опциональный** (резервный путь), если прямой запрос к YouTube API из браузера не сработал в вашей сети/браузере.

Запуск (если нужен):

```bash
node tools/local-youtube-parser.js
```

По умолчанию адрес:
- `http://127.0.0.1:8787/youtube?url=...`

В профиле MyTube поле «Локальный URL парсера» тоже опциональное.

## Что импортируется

- просмотры;
- лайки;
- до 50 верхнеуровневых комментариев с никнеймами авторов;
- до 10 первых ответов к каждому комментарию (если ответы доступны через API);
- превью, название и канал.

## Если всё же хотите запускать локальный парсер

Опциональные переменные окружения:

- `MYTUBE_PARSER_HOST` (по умолчанию `127.0.0.1`)
- `MYTUBE_PARSER_PORT` (по умолчанию `8787`)
- `MYTUBE_YOUTUBE_API_KEY` (переопределение ключа для сервера)

Пример:

```bash
MYTUBE_PARSER_PORT=8899 MYTUBE_YOUTUBE_API_KEY=your_key node tools/local-youtube-parser.js
```


## Автопарсинг всей библиотеки

В профиле нажмите **«Автопарсить всю библиотеку»**.
Приложение само:
- найдёт видео без превью/комментариев/статистики;
- выполнит поиск YouTube по названию (с нормализацией);
- подгрузит просмотры, лайки, комментарии и ответы для совпадений;
- скачает JSON-файл с низкой уверенностью совпадений для ручной проверки.

Во время процесса показывается прогресс-бар.
